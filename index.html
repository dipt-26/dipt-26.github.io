Abstract

Retrieving a target object from dense tabletop clutter is a challenging long-horizon manipulation problem due to tight packing, complex contact dynamics, and the combinatorial explosion of possible push sequences. Traditional planning-based systems rely on parallelized Monte Carlo Tree Search (MCTS), but suffer from heavy computational overhead, often requiring 25–30 seconds per action.

We present a fast and reactive reinforcement learning alternative based on Proximal Policy Optimization (PPO). Our agent learns 16 discrete, quasi-static planar push primitives and uses a carefully designed reward scheme to efficiently create graspable clearances in cluttered scenes—achieving real-time decision making without expensive tree search.

Despite being trained from scratch, the learned policy generalizes strongly, retrieving targets in heavily packed environments with ≈92% success and handling up to 11 objects. The full retrieval process completes within ~20 seconds per scene, offering a >10× speedup over state-of-the-art MCTS-based approaches while maintaining comparable or superior performance.

Our Pipeline

Our continuous non-prehensile manipulation pipeline (CPRL) integrates perception, control, and grasp prediction into a real-time framework for retrieving a target object from dense clutter. A dual-camera eye-on-hand setup provides uninterrupted 3D pose estimates (centers + in-plane rotations) of all objects, even during pushing. These observations feed into a PPO-based control module, where the RL policy selects one of 16 push primitives to gradually create graspable clearance around the target. The policy operates in simulation using object bounding boxes and gripper state, optimized with a structured reward encouraging reduced clutter overlap and improved target isolation. Once the scene becomes graspable, our accelerated grasp network computes the final top-down grasp configuration (X, Y, θ) using GPU-batched inference for a 10× speedup. Together, these components enable continuous pushing, real-time perception, and fast grasp execution without any stop–sense–plan delays.
